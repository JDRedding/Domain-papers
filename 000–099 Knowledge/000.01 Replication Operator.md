## 000.01 — The Replication Operator 𝓡
### *A formal RDG‑aligned definition of replicability in scientific knowledge systems*  
*(Draft for the 000–099 Knowledge domain)*

---

## **Abstract**  
The replication crisis across social and behavioural sciences reveals a structural instability in how scientific claims propagate across independent reconstructions. This paper defines the **Replication Operator 𝓡** as a formal epistemic transformation acting on studies, claims, and measurement operators. Using the RDG triad (SID/PED/Q), we characterize replicability as **operator invariance** under reconstruction. We show that replication failure emerges when contextual, methodological, or theoretical components drift between original and replication operators. The Replication Operator 𝓡 provides a generalizable framework for analyzing, predicting, and improving replicability across all scientific domains.

---

## **1. Introduction**  
Replicability is often treated as a statistical property of studies, but it is more fundamentally an **operator property** of knowledge systems. A study replicates when an independent reconstruction of its structure, interaction rules, and measurement dynamics yields a consistent claim.

In RDG terms:

```
Replication succeeds when:   𝓡(Study) ≈ 𝓞(Study)
Replication fails when:      𝓡(Study) ≠ 𝓞(Study)
```

Where:

- **𝓞** = Original operator  
- **𝓡** = Replication operator  

This paper defines 𝓡 formally and shows how it interacts with SID/PED/Q.

---

## **2. The RDG Triad and Replication**

### **2.1 SID — Structure, Interaction, Dynamics**  
SID defines the *phenomenon* being studied:

- **Structure:** latent variables, constructs, causal graphs  
- **Interaction:** contextual rules, incentives, framing  
- **Dynamics:** timing, feedback, sequence of events  

Replication requires **SID invariance**:

```
SID_original ≈ SID_replication
```

### **2.2 PED — Power, Evaluation, Dynamics (of measurement)**  
PED defines the *method* used to study the phenomenon:

- **Power:** statistical sensitivity  
- **Evaluation:** analytic operators, model families  
- **Dynamics:** measurement timing, exclusion rules, transformations  

Replication requires **PED invariance**:

```
PED_original ≈ PED_replication
```

### **2.3 Q — Query / Claim**  
Q defines the *statement* made about the phenomenon.

Replication requires **claim stability**:

```
Q_original ≈ Q_replication
```

---

## **3. Formal Definition of the Replication Operator 𝓡**

Let a study be represented as:

```
Study = SID ∘ PED ∘ Q
```

Then the replication operator is:

```
𝓡(Study) = SID' ∘ PED' ∘ Q'
```

Replication succeeds when:

```
SID' ≈ SID
PED' ≈ PED
Q'  ≈ Q
```

Replication fails when any component diverges.

---

## **4. Sources of Divergence**

### **4.1 Contextual Drift (SID failure)**  
Changes in:

- population  
- incentives  
- framing  
- environment  

cause SID' ≠ SID.

### **4.2 Methodological Drift (PED failure)**  
Changes in:

- analytic flexibility  
- exclusion criteria  
- transformations  
- model selection  

cause PED' ≠ PED.

### **4.3 Claim Drift (Q failure)**  
Changes in:

- effect size  
- statistical significance  
- interpretation  
- causal attribution  

cause Q' ≠ Q.

---

## **5. The Replication Equation**

We define replication probability **P_rep** as:

```
P_rep = P(SID' ≈ SID) × P(PED' ≈ PED) × P(Q' ≈ Q)
```

This yields a **multiplicative fragility**:

- If any component is unstable, replication collapses.
- Soft sciences have lower SID and PED invariance → lower P_rep.

This matches bibliometric evidence for the **Hierarchy of the Sciences**.

---

## **6. Hardness Gradient and Replication Stability**

Using Fanelli & Glänzel’s hierarchy:

```
Hard → Soft
500 → 600 → 300 → 400 → 800 → 900
```

We define hardness **H** as:

```
H = P(SID invariance) + P(PED invariance)
```

Then:

```
P_rep ∝ H
```

This explains:

- high replication in physics  
- moderate replication in biology  
- low replication in psychology  
- very low replication in social sciences

## **7. Practical Implications**

### **7.1 For researchers**  
Increase replication probability by:

- stabilizing SID (context control)  
- stabilizing PED (analytic preregistration)  
- stabilizing Q (precise claims)

### **7.2 For journals**  
Require:

- preregistered operators  
- full materials  
- full analytic pipelines  
- multi-lab validation

### **7.3 For meta-science**  
Use 𝓡 to:

- classify fields by replicability  
- predict effect-size shrinkage  
- detect operator drift  
- design replication-ready studies

## **9. Conclusion**  
The Replication Operator 𝓡 provides a formal, domain‑agnostic definition of replicability. It reveals that replication is not merely statistical but structural: a study replicates when its operators remain invariant under reconstruction. This framework unifies replication theory across all scientific domains and provides actionable guidance for improving scientific reliability.
